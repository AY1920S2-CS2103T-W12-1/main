Cher Wei Jie UML Diagrams

  • Return Features UML Diagrams

This Activity Diagram purpose is to describe the general work flow of the Return feature.

This Class Diagram purpose is to describe the dependencies of ReturnCommand class when implementing this feature. It allows a developer to easily see the classes and interfaces associated with the ReturnCommand.

The first Sequence Diagram depicts a simple overview of the entire execution flow of a valid return command for converting a delivered order into a return order. The second Sequence Diagram depicts a simple overview of the entire execution flow of a valid return command for creating a new return order based on given parcel attributes.

The third Sequence Diagram depicts a detailed execution flow of a valid return command for converting a order into a return order. The fourth Sequence Diagram depicts a detailed execution flow of a valid return command for creating a new return order and this will throw an exception and display an error message to the user if any compulsory parcel attribute is missing.

  • Delivered Features UML Diagram

This Activity Diagram purpose is to describe the general work flow of the Delivered feature.

This Class Diagram purpose is to describe the dependencies of DeliveredCommand class when implementing this feature. It allows a developer to easily see the classes and interfaces associated with the DeliveredCommand.

This Sequence Diagram depicts a simple overview of the entire execution flow of a valid delivered command for marking an order or return order as delivered.

This Sequence Diagram depicts a detailed execution flow of a valid delivered command for delivering an order and this will throw an exception and display an error message to the user if the FLAG and/or the INDEX is/are invalid or missing. If both FLAG and INDEX are valid, the delivered command will update the respective order or return order list (depending on the provided FLAG) and display the success message to the user in the GUI.

Please refer to the following link for the implementation of the class diagram, sequence diagram and activity diagram of Help Command. https://ay1920s2-cs2103t-w12-1.github.io/main/DeveloperGuide.html#help
